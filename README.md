# Ramda Validations

JavaScript type testing library for `ramda`

## Validators List

### General

- isPresent
- isBlank
- isOtherThan

### Number

- isEven
- isOdd
- isNegative
- isPositive
- isInteger
- isFloat
- isFinite
- isMultiple

## String

- isAlphanum
- isString

## Boolean

- isTrue
- isFalse

## Array

- isInRange

### Object

- hasKeys
- isObject
- isHash

### Promise

- isPromise
